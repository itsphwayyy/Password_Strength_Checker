# Password_Strength_Checker
# How this work?

If there are 2 or more character, +2 marks.
If there is 1 character, +1 mark.
If 0, no mark. 

If there are 2 or more digit, +2 marks.
If there is 1 digit, +1 mark.
If 0, no mark. 

If there are 2 or more specific character, +2 marks.
If there is 1 specific character, +1 mark.
If 0, no mark. 

If  you get 6 marks, you have a perfect password.
If your mark is 3 or more than 3, you have medium strong password.
If your mark is less than 3, it is a weak password.
